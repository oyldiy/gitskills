1，创建版本库
仓库名	repository 简单理解为目录。目录里面所有的文件可以被git管理起来。每个文件的修改，删除，git都可以跟踪，以便任何事可以可以追踪历史版本，或者将来某个时刻还原。


mkdir gitck
cd gitck


git init 
这个命令把当前目录变成git可以管理的仓库了。

工作区和暂存区
我们电脑的目录是工作区。


git 自动创建的 第一个分支  master 指针HEAD
git add 添加文件 就是添加到暂存区。
git commit 就是提交到当前分支。
